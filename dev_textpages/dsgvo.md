## 3. KI-Funktion (Express Creation)

Die Anwendung bietet eine optionale KI-gestützte Funktion zur automatischen Klassifizierung von Ausgaben. Die Funktion steht auf zwei Wegen zur Verfügung:

**a) Geteilter öffentlicher API-Schlüssel**
Sofern vom Betreiber ein geteilter API-Schlüssel bereitgestellt wird, kann die KI-Funktion ohne eigenen Schlüssel genutzt werden. In diesem Fall gilt zusätzlich:
- Der Umfang der KI-Nutzung wird je Nutzer intern erfasst und gespeichert (verbrauchte API-Kosten in Cent), um ein pro Nutzer geltendes Kostenlimit durchzusetzen.

**b) Eigener API-Schlüssel**
Nutzer können in den Einstellungen einen eigenen Anthropic-API-Schlüssel hinterlegen. In diesem Fall wird kein Nutzungsumfang intern erfasst.

**Weitergabe an Dritte (gilt für beide Varianten)**
Die zur Klassifizierung eingegebenen Texte und Medien werden an die API von Anthropic, Inc. (San Francisco, USA) übermittelt und dort verarbeitet. Es gelten die [Datenschutzbestimmungen von Anthropic](https://www.anthropic.com/privacy).
