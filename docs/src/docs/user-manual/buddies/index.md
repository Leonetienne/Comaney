# Buddies

Buddies lets you track shared spending with another person. Whether you are splitting a restaurant bill, covering each other's grocery runs, or settling up after a weekend away, Comaney keeps a running tally of who owes who so you never have to work it out yourself.

## How it works

Whoever pays upfront logs the full expense and marks which people share the cost. Each person's share is recorded automatically. Everyone's balance stays up to date.

## One-on-one buddies

Buddies are for splitting costs with a single person. You can add someone as an offline buddy (they do not need a Comaney account) or connect with another Comaney user by sending them an invitation.
